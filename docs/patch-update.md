# Lightweight Patch Update

## Goal
Ship small updates without rebuilding and distributing the full installer.

## Artifacts
- `you2midi-updater.exe` (bundled with desktop release)
- Patch ZIP (generated by `scripts/build_patch.ps1`)

## Build Patch
```powershell
make patch-build
```

Default patch contents:
- `You2midi.exe`
- `launcher-config.json`
- `you2midi-desktop.exe`
- `you2midi-backend.exe`
- `you2midi-updater.exe`
- `runtime/python-runtime.json` (if present)

Output:
- `dist/patch/you2midi-patch-<version>.zip`

## Apply Patch (manual)
From installed app directory:
```powershell
.\you2midi-updater.exe -patch "C:\path\to\you2midi-patch-dev.zip" -app-dir .
```

Or from URL:
```powershell
.\you2midi-updater.exe -patch "https://example.com/you2midi-patch-dev.zip" -app-dir .
```

## App Integration
Desktop app exposes `StartPatchUpdate(patchRef)` for one-click update flows:
- Starts updater process
- Waits for current app process to exit
- Applies patch
- Relaunches desktop app
